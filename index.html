Claude Windows 설치 다운로드 가이드

Windows 환경에서 Claude를 설치하고 사용하는 방법을 안내합니다.

1. 시스템 요구 사항

항목요구 사양
운영체제Windows 10 (64bit) 이상 / Windows 11 권장
RAM최소 4GB, 권장 8GB 이상
디스크 공간약 500MB 이상
네트워크인터넷 연결 필수

2. Claude Desktop 앱 설치 (일반 사용자)

  1. 공식 사이트 claude.ai/download 접속
  2. Download for Windows 클릭하여 .exe 다운로드
  3. 설치 파일 더블 클릭 → SmartScreen 경고 시 추가 정보 → 실행
  4. 설치 마법사 진행 후 Anthropic 계정으로 로그인
Claude Desktop 다운로드 웹에서 바로 사용

3. Claude Code 설치 (개발자용 CLI)

3-1. Node.js 설치

nodejs.org에서 LTS 버전 다운로드 후 설치. PowerShell에서 확인:

node --version
npm --version

3-2. Claude Code 전역 설치

npm install -g @anthropic-ai/claude-code

작업 폴더로 이동 후 실행:

cd C:\Users\사용자명\프로젝트
claude

4. 설치 확인

claude --version
claude --help

5. 자주 발생하는 문제

SmartScreen 차단 — 추가 정보 → 실행으로 진행
npm 권한 오류 — PowerShell을 관리자 권한으로 실행하거나 npm config set prefix %APPDATA%\npm
로그인 실패 — 프록시/방화벽이 anthropic.com을 차단하는지 확인
